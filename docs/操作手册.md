## 1. 快速开始

推荐流程：

```
下载模板 → 填写 Excel → 上传两份数据 → 设置规则 → 自动/手动排班 → 导出 Excel
```

左侧导航栏六个入口：

| 菜单 | 作用 |
|------|------|
| 仪表盘 | 上传数据、下载模板 |
| 排班编辑 | 全局日历/矩阵分配 |
| 字幕员表 | 查看人员可用性 |
| 电影排片 | 查看排片结构 |
| 排班总表 | **主编辑 + 导出** |
| 设置 | 行政区优先级、排班约束 |

## 2. Step 1：下载并填写标准模板

打开 **仪表盘**，在「数据导入」卡片底部有两个下载按钮。

![仪表盘 — 数据导入与模板下载](./screenshots/01-dashboard.png)

### 2.1 下载字幕员模板

1. 点击金色按钮 **「下载字幕员模板」**
2. 浏览器将下载：`后五天问卷调查 给静静.xlsx`
3. 用 Excel 打开，填写或核对：
   - **姓名**列
   - **区域**列（负责哪个区填 1）
   - **搭档**列
   - 各 **日期-时间段** 列（空白/null = 有空，有内容 = 已占用）

### 2.2 下载排片表模板

1. 点击灰色按钮 **「下载排片表模板」**
2. 浏览器将下载：`2024排片表.xlsx`
3. 用 Excel 打开，填写或核对：
   - **日期、周、影院、影厅** 列
   - 各 **时间段** 列（08:30 ~ 20:40）的电影信息

> **提示**：模板文件与系统解析逻辑完全匹配，建议直接基于模板修改，避免表头格式不兼容。

## 3. Step 2：上传数据

仍在仪表盘页面：

![仪表盘 — 已上传状态示例](./screenshots/01-dashboard.png)

### 3.1 上传字幕员数据

1. 点击 **「字幕员数据」** 标签（默认选中）
2. 将填好的 `后五天问卷调查 给静静.xlsx` 拖入虚线框，或点击选择文件
3. 解析成功后，底部状态栏显示绿色 ✓ 及文件名
4. 弹出提示时可选择跳转到「字幕员表」查看

### 3.2 上传电影排片表

1. 点击 **「电影排片表」** 标签
2. 上传 `2024排片表.xlsx`
3. 解析成功后，底部显示文件名
4. 可选择跳转到「电影排片」查看

### 3.3 对排班总表的影响

| 上传内容 | 总表变化 |
|----------|----------|
| 排片表 | 重建所有 **cells**（场次行），已有分配尽量保留 |
| 字幕员表 | 不改变总表结构，仅影响后续**可选字幕员** |

## 4. Step 3：核对电影排片

进入 **电影排片** 页面：

![电影排片表](./screenshots/02-schedule-table.png)

### 操作说明

| 区域 | 操作 |
|------|------|
| 顶部日期标签 | 点击切换日期（如 6月14日 五） |
| 左侧影院列表 | 点击选择影院 |
| 右侧表格 | 查看 影厅 × 时间段 的电影信息 |
| 底部统计 | 查看影院数、场次数 |
| 上传/更新数据 | 返回仪表盘重新上传 |

**与总表关系**：表格中有绿色内容的格子 = 排班总表中对应场次需要分配 2 名字幕员。本页只读，不能直接改排班。

---

## 5. Step 4：核对字幕员可用性

进入 **字幕员表** 页面：

![字幕员时间表](./screenshots/03-subtitler.png)

### 操作说明

| 区域 | 操作 |
|------|------|
| 顶部日期标签 | 切换 18日、19日… 查看不同日期 |
| 区域列 | 绿色「1」= 负责该区域 |
| 时间段列 | 绿色 = 有空（可分配），有字 = 已占用 |
| 搭档列 | 固定搭档，自动排班时优先配对 |

**与总表关系**：总表分配时的下拉列表，只显示此处为「绿色空闲」的字幕员。若下拉为空，先回到此页确认该时段是否有人有空。

---

## 6. Step 5：配置排班规则（可选）

进入 **设置** 页面：

![设置 — 行政区优先级](./screenshots/06-settings.png)

### 6.1 行政区优先级

- 拖动列表调整 11 个区的顺序
- 越靠前，自动排班时越先处理
- 点击 **保存设置** 生效

### 6.2 排班大前提条件

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 每天排班限制 | 2 场/天 | 自动排班生效 |
| 总场次限制 | 5 场/总计 | 自动排班生效 |
| 同一时间段互斥 | 开启 | 同一人同时只能去一个厅 |
| 优先时间块多的字幕员 | 开启 | 优先安排剩余时段少的人 |

**与总表关系**：仅影响 **自动排班** 的分配结果，手动下拉不受这些上限强制拦截。

## 7. Step 6：排班编辑（日历/矩阵）

进入 **排班编辑** 页面：

![排班编辑 — 日历视图](./screenshots/04-schedule-edit.png)

### 7.1 工具栏

| 按钮 | 功能 |
|------|------|
| 切换至矩阵视图 | 日历 ↔ 表格两种布局 |
| 手动排班 | 启用下拉编辑 |
| 只读模式 | 禁止修改，仅查看 |
| 进度统计 | 共 N 场 · 已完成 M 场 |

### 7.2 日历视图

- **左侧**：字幕员列表，可 **拖拽** 到右侧场次 → 快速填入字幕员1
- **右侧**：按日期分组的场次卡片
- 每场卡片有两个下拉：**字幕员1**（蓝）、**字幕员2**（绿）
- 双人填完 → 卡片变绿；冲突 → 变红

### 7.3 矩阵视图

表格列：日期/时间 | 影片 | 字幕员1 | 字幕员2，适合逐行批量操作。

**与总表关系**：此处每次选择/拖拽都会 **实时写入排班总表**（`assignments`），与「排班总表」页完全同步。

## 8. Step 7：排班总表（主工作区）

进入 **排班总表** 页面 — 这是最终核对与导出的主界面：

![排班总表](./screenshots/05-schedule-overview.png)

### 8.1 头部操作

| 控件 | 说明 |
|------|------|
| 手动排班 | 表格内出现可编辑下拉 |
| 自动排班 | 切换为只读 + 显示「执行自动排班」按钮 |
| 清除 | 清空所有已分配字幕员（需确认） |
| 导出 Excel | 下载完整排班结果 |
| 执行自动排班 | 按规则批量填充（保留已有手动分配） |

### 8.2 主体布局

与「电影排片」类似，但表格多了两列：

| 列 | 可编辑 | 写入总表字段 |
|----|--------|-------------|
| 影厅 | 否 | — |
| 时间 | 否 | — |
| 电影 | 否 | — |
| 字幕员1 | 是（手动模式） | assignments[key].subtitler1 |
| 字幕员2 | 是（手动模式） | assignments[key].subtitler2 |

- 顶部日期标签 + 左侧影院列表筛选当前视图
- 分配完成行显示绿色背景
- 底部提示：「下拉列表只显示此时段有空且未被占用的字幕员」

### 8.3 推荐操作顺序

1. 切换 **自动排班** → 点击 **执行自动排班**
2. 查看头部进度「已分配 X / Y 场」
3. 切换 **手动排班** → 逐影院补全未分配场次
4. 全部完成后 → **导出 Excel**

---

## 9. Step 8：导出结果

在排班总表页点击 **「导出 Excel」**：

- 文件名：`排班总表_YYYYMMDD.xlsx`
- 每个 festival 日期一个 Sheet
- 列：影院 | 影厅 | 日期 | 星期 | 时间 | 电影 | 字幕员1 | 字幕员2
- 未识别区域的影院行以灰色标记

导出文件即为可交付的最终排班总表。

## 10. 常见问题

### Q：模板下载没反应？

- 确认使用 Chrome / Edge / Safari 等现代浏览器
- 检查是否被浏览器拦截下载
- 开发环境需确保 `public/templates/` 下两个 xlsx 文件存在

### Q：上传后排班总表是空的？

- 需先上传 **电影排片表**，系统才能生成 cells
- 仅上传字幕员表不会产生场次

### Q：下拉列表显示「此时段无空闲字幕员」？

1. 到字幕员表确认该日期、时段是否有绿色格子
2. 确认是否已被同场另一字幕员选中
3. 确认是否已上传字幕员数据

### Q：排班编辑和排班总表有什么区别？

- **排班编辑**：全局浏览，支持拖拽，适合宏观分配
- **排班总表**：按影院逐日核对，支持自动排班和导出
- 两者操作同一份数据，互通同步

### Q：刷新页面数据会丢吗？

不会。数据保存在浏览器 localStorage。清除浏览器缓存会导致丢失。
